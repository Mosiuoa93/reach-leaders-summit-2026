## ğŸ“ Need Help?

### Quick Questions

**Q: What format should my logo be?**
A: SVG is best (scalable), but PNG or JPG work too.

**Q: Where do I put the logo file?**
A: `frontend/public/` folder

**Q: Do I need to change any code?**
A: Only if using PNG/JPG instead of SVG.

**Q: How do I adjust the size?**
A: Edit `.logo` class in `App.css`

**Q: How do I deploy the changes?**
A: Push to GitHub, Vercel auto-deploys.

### Troubleshooting

**Logo not showing?**
- Check file is in `frontend/public/`
- Hard refresh browser (Ctrl+Shift+R)
- Restart frontend (npm start)

**Logo distorted?**
- Make sure width = height in CSS
- Check `object-fit: contain;` is set

**Logo wrong size?**
- Edit width and height in `App.css`
- Make sure both are equal

**Code won't update?**
- Check filename in LandingPage.js
- Make sure it matches your file
